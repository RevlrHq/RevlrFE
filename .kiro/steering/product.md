# Product Overview

Revlr is an event management platform that enables organizers to create, manage, and analyze events. The platform provides comprehensive tools for:

- Event creation and publishing
- Ticket management and sales
- Attendee registration and management
- Real-time analytics and performance tracking
- Revenue reporting and payout management
- Media search and management for event assets
- Organizer dashboard with comprehensive event oversight

The platform serves event organizers with features ranging from basic event setup to advanced analytics, attendee management, and revenue optimization. It includes both organizer-facing tools and attendee-facing registration/ticketing flows.

Key user types:

- Event organizers (primary users)
- Event attendees
- Vendors/partners with special access
